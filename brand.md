# Kuentas Klaras - Brand Guidelines v1.0

## Concepto

Kuentas Klaras es una aplicación de control financiero personal enfocada en la simplicidad, claridad y control.

La identidad visual se basa en una “K” geométrica construida mediante formas triangulares que representan:

- Ingresos

- Gastos

- Ahorro

- Crecimiento

La convergencia de las formas simboliza el control y la organización financiera.

# Estilo Visual

La interfaz debe transmitir:

- Limpieza

- Simplicidad

- Modernidad

- Tecnología

- Seguridad financiera

Evitar:

- Colores agresivos

- Exceso de sombras

- Gradientes fuertes

- Elementos recargados

Priorizar:

- Espacios amplios

- Jerarquía visual clara

- Componentes minimalistas

- Experiencia intuitiva
